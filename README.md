# 🍲 Recipe Management System

A web-based Recipe Management System built using Django that allows users to create, manage, and explore recipes with authentication support.

---

## 🚀 Features

- 🔐 User Authentication (Register & Login)
- ➕ Add new recipes
- ✏️ Update existing recipes
- ❌ Delete recipes
- 🔍 Search recipes
- 📋 View all recipes in a clean UI

---

## 🛠️ Tech Stack

- **Backend:** Django (Python)
- **Database:** SQLite
- **Frontend:** HTML, CSS, Bootstrap
- **Authentication:** Django Auth System
